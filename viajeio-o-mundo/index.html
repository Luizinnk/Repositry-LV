Sobre o projeto

Trabalho final de Desenvolvimento Web

Escolhi a Copa do Mundo 2026 como tema porque é algo que todos conhecem — e transformei isso num exercício completo de front-end. Cada página de país é gerada por JavaScript, as fotos são buscadas em tempo real pela API do Wikimedia Commons, e o design foi todo construído em CSS moderno, sem nenhum framework.
